# Jarvis 1.1 desktop shell

## Goal

Replace the inherited multi-panel OpenJarvis interface with the focused Jarvis
assistant screen.

## Scope

- One black, voice-first chat screen: orb, conversation area, text input and
  microphone control.
- Keep camera activation intent-driven; it has no visible launcher.
- Remove visible navigation, model selection, conversation history, system
  metrics and the Dashboard, Data Sources, Agents and Logs routes.
- Switch the Windows NSIS installer to a per-user installation so subsequent
  updates do not require administrator elevation.
- Publish this release as 1.1.0.

## Deliberate limits

Settings remains out of the primary shell until its replacement is designed.
The removed modules can remain in source temporarily, but are no longer routed
or imported by the desktop shell, so they do not ship in the frontend bundle.
