# JarvisAI

Asistente de escritorio para Windows, activado por voz y diseñado para vivir
en segundo plano. JarvisAI combina una interfaz conversacional, esfera neural,
voz, cámara y automatización local en una aplicación nativa.

## Instalación

1. Descarga `Jarvis_*_x64-setup.exe` desde la última
   [release de JarvisAI](https://github.com/Machi2603/JarvisAI/releases).
2. Ejecuta el instalador y abre **Jarvis**.
3. Añade tu clave de Groq durante la configuración inicial.
4. Cuando termine la descarga inicial de los modelos de voz, di **"Jarvis"**
   seguido de tu petición.

No necesitas instalar Docker, Python, Node.js, Rust, Ollama ni herramientas de
línea de comandos.

## Actualizaciones

Las versiones estables comprueban actualizaciones al iniciar la aplicación y
cada 30 minutos. Las descargas se verifican criptográficamente antes de la
instalación. Publicamos las nuevas versiones en
[GitHub Releases](https://github.com/Machi2603/JarvisAI/releases).

## Privacidad

Las claves de proveedores cloud se almacenan en el almacén de credenciales de
Windows. El detector de palabra de activación y las funciones de voz locales se
ejecutan en tu equipo.
